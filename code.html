The Code for Rewind


                        //Get the string from the page, controller function
function getString(){
    document.getElementById("alert").classList.add("invisible");

    let userString = document.getElementById("userString").value;
    let reversedString = reverseString(userString);

    displayString(reversedString);
}

//Reverse the string, logic function
function reverseString(userString){
    let revString = [];

    //Reverse the string with a decrement for loop
    for (let i = userString.length - 1; i >= 0; i--){
        revString += userString[i]; //Not the array.push() method
    }
    return revString;
}

//Display the reversed string and message to the user, view function
function displayString(reversedString){
    //Write message to the page
    document.getElementById("message").innerHTML = `The reversed string is: ${reversedString}`;

    //Show the alert box
    document.getElementById("alert").classList.remove("invisible");
}

The code is structured in three functions, getString(), reverseString() & displayString()

getString()

This function is triggered by an event listener document.getElementById("btnSubmit").addEventListener("click", getValues when the 'Reverse' button is clicked in the app page. The function first adds the class 'invisible' to a div with the id 'alert'. This makes sure that this div is initially hidden from the user. This is achieved through the method document.getElementById("alert").classList.add("invisible"); The function then creates two variables, userString whose value is passed from the document.getElementById("userString").value method and a reversedString variable whose value will be calculated through the reverseString() method. This variable is then used as an argument to display the final output to the user with the displayString() method.

reverseString()

This function takes a parameter, userString. The function initializes an empty array called revString which will hold the reversed string. Using a decrement for loop, we start to loop through the user's string. The starting point for the counter is set as userString.length - 1, this starts the loop from the last character of the user's string. The need to subtract 1 from the length of the users's string is because indexes start from 0. The loop runs until the counter is at the start index, 0, i.e. i >= 0, decrementing the value of i by 1 on each loop through i-- On each loop, the function adds (+=) each of the character of the user's string in the reverse order. Here it is important that we do not push the characters as the array will store the characters individually with their own indexes. Finally the function returns the reversed string as an array.

displayString()

The parameter for this function is defined as reversedString. If we go back to the getString() method, we are passing the now reversed string (output of the reverseString() function) as an argument to this function. Finally, through DOM manipulation, the result is passed to the paragraph element which has an id of "message". This is achieved through the document.getElementById("message").innerHTML = `The reversed string is: ${reversedString}`; method. This function also removes the "invisible" class from the div with the id of alert which makes reversed string visible to the user.
